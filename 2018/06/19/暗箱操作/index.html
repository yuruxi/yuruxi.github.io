暗箱操作

发表于 2018-06-19

首先安装Hexo，在盘里随便创建一个文件夹。然后输入d: cd blog，随后开始操作。输入npm install hexo -g，开始安装Hexo；输入hexo -v检查hexo是否安装成功；输入hexo init初始化该文件夹，然后就会看到最后有一句：Start blogging with Hexo，说明我们配置成功！接着输入npm install安装所需要的组件，然后输入hexo g，再输入hexo s，就可以开启服务器出现localhost:4000，就可以正式体验Hexo。
将Hexo和Github page联系起来，用到的代码是git config –global user.name “exm”和git config –global user.email “573858531@qq.com“，然后输入cd ~/.ssh，检查是否有.ssh文件夹，输入ls就会列出文件下的内容；输入ssh-keygen -t rsa-C”573858531@qq.com“，连续三个回车，生成的密钥，得到两个文件，输入eval “$(ssh-agent -s)，添加密钥到ssh-agent，再输入ssh-add ~/.ssh/id_rsa，添加生生的SSH key到ssh-agent。然后，注册GitHub账号并登陆，点击settings添加ssh，新建一个new ssh key，把id_rsa.pub的内容复制上去。输入ssh -T git@github.com，测试添加ssh是否成功。新建一篇博客，在cmd执行命令：hexo new post “博客名”，在生成以及部署文章之前，需要安装一个扩展：npm install hexo-deployer-git –save，用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了，用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了。
