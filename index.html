暗箱操作

发表于 2018-06-19

首先安装Hexo，在盘里随便创建一个文件夹。然后输入d: cd blog，随后开始操作。输入npm install hexo -g，开始安装Hexo；输入hexo -v检查hexo是否安装成功；输入hexo init初始化该文件夹，然后就会看到最后有一句：Start blogging with Hexo，说明我们配置成功！接着输入npm install安装所需要的组件，然后输入hexo g，再输入hexo s，就可以开启服务器出现localhost:4000，就可以正式体验Hexo。
将Hexo和Github page联系起来，用到的代码是git config –global user.name “exm”和git config –global user.email “573858531@qq.com“，然后输入cd ~/.ssh，检查是否有.ssh文件夹，输入ls就会列出文件下的内容；输入ssh-keygen -t rsa-C”573858531@qq.com“，连续三个回车，生成的密钥，得到两个文件，输入eval “$(ssh-agent -s)，添加密钥到ssh-agent，再输入ssh-add ~/.ssh/id_rsa，添加生生的SSH key到ssh-agent。然后，注册GitHub账号并登陆，点击settings添加ssh，新建一个new ssh key，把id_rsa.pub的内容复制上去。输入ssh -T git@github.com，测试添加ssh是否成功。新建一篇博客，在cmd执行命令：hexo new post “博客名”，在生成以及部署文章之前，需要安装一个扩展：npm install hexo-deployer-git –save，用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了，用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了。

“鎖那”

发表于 2018-06-13

sana是以动画投稿网站为中心活动中的女性歌手。
以让人感觉身处在温暖阳光中，柔和而富有透明感的歌声为特征。
制造出了仅仅是用治愈表达不出来的奇幻世界观。
2010年1月，中学三年级时期在niconico动画上首次以「锁那」的名义投稿了翻唱动画。
同年7月开始生放送。
2011年11月，投稿HoneyWorks的曲目[初恋的绘本]翻唱动画，一口气夺下10万播放数。
从此开始恒定的投稿以HoneyWorks乐曲为主的翻唱歌曲，包括播放数超过50万，mylist（收藏）超过21000的「告白予行练习　ver.锁那」在内总数超过50曲。
生放送目前为止放送次数超过1100次，生放社区人数将要超过40000人。
在同人活动方面不仅发布了自主制作的cd并且达到每张5000以上的销量。
随后2015年4月以[sana]的名义，经由HoneyWorks制作担当tv动画[火影忍者疾风传]ed曲。
5月作为出道单曲发售：sana/HoneyWorks[不需要语言的约定]。
同年7月参加HoneyWorks 3rd专辑[喜欢上你的瞬间]。
之后担任HoneyWorks初次东阪one man live嘉宾歌手。

“羽入”

发表于 2018-06-13

和梨花一样自称为“ボク”以及有“的说”（～なのです）的口癖，除此之外兴奋、害怕、紧张经常会“啊呜啊呜”（あぅあぅ）地叫起来。
性格很容易放弃，屈服于命运，对于”命运”的态度十分冷淡、但是非常害怕梨花由于”命运”而在精神上死亡。在梨花身边的态度略显消极，一度认为梨花永远无法逃出这个死胡同。提到雏见泽的奇怪现象－－”オヤシロさまの祟り”会很心痛。
原本认为自己是除了帮助梨花轮回，无法影响任何事的无力的存在。因此一直在为自己的无能向主角们不断道歉，不停地说对不起。
喜欢跟在她关注的人背后，经常，因此在无意中间接加快了不少悲剧的发生。雏见泽症候群发作的人可以隐约感受到其存在，主角们听到的脚步声、道歉声、身后感到的气息等就是她的所为。
尽管性格较为消极，但在面对鹰野三四等敌人时有着十分严肃的一面，会毫不犹豫地选择保护雏见泽
